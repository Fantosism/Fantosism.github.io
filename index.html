Work I have done.

Helpfull

The Helpfull MVP was released after a 3 week deadline. The project was done on a team of five following an Agile-Scrum methodology under a Senior Developer. Held weekly sprint introspectives and launched on Product Hunt on day 15. Successfully learned and implemented React hooks for state management. JSX was used in addition to CSS for markup. Node and Express were used to build a RESTful API for the backend, with MongoDB being chosen as the storage solution for persistent data. Authorization is done with JWT using Passport strategies. Testing done with Mocha/Chai on backend and Enzyme/Jest on client. Helpfull was made with the intent to ease the difficulty in locating volunteer opportunities closest to you. We query the Google Maps API in order to filter everything by proximity to your current location.

Tools Used: Javascript, React (hooks), MongoDB/Mongoose, JWT, bCrypt, Passport, Express, Node, React-Google-Maps, Mocha/Chai, Jest/Enzyme

Easy as Python

Easy as Python was created to implement an alogorithm with a singly linked list in order to practice spaced repetition with the Python programming language. Registering for the App allows the user to login to store their progress and get a score depending on how many questions they have answered correctly. There's also a limited trial mode for learning without an account. This application was created with React and Redux for state management, with Node and Express for the backend. MongoDB was used for persistent data, JWT's for authentication with Mocha/Chai and Enzyme for back/front-end testing respectively.

Tools Used: Javascript, React, Redux, MongoDB/Mongoose, JWT, bCrypt, Passport, Express, Node, Mocha/Chai, Enzyme

MyMDB

MyMDB was created to be an online movie database. A user can query the Api for a movie of their choice, view a trailer, and add to a watch or favorite list. The user must register an account before being able to use the application. Future versions will plan on having streaming sites the movie is viewable on persisted in the database as well. Used the OMDB and TMDB API.

Tools Used: Javascript, React(hooks), Redux, MongoDB/Mongoose, JWT, bCrypt, Passport, Express, Node
