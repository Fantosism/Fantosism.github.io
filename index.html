WORK I HAVE DONE

Helpfull

The Helpfull MVP was released after a 3 week deadline. The project was done on a team of five following an Agile-Scrum methodology under a Senior Developer. Held weekly sprint introspectives and launched on Product Hunt on day 15. Successfully learned and implemented React hooks for state management. JSX was used in addition to CSS for markup. Node and Express were used to build a RESTful API for the backend, with MongoDB being chosen as the storage solution for persistent data. Authorization is done with JWT using Passport strategies. Testing done with Mocha/Chai on backend and Enzyme/Jest on client. Helpfull was made with the intent to ease the difficulty in locating volunteer opportunities closest to you. We query the Google Maps API in order to filter everything by proximity to your current location.

Tools Used: Javascript, React (hooks), MongoDB/Mongoose, JWT, bCrypt, Passport, Express, Node, React-Google-Maps, Mocha/Chai, Jest/Enzyme
